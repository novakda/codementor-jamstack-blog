# Requirements: JAMstack Blog Tutorial

**Defined:** 2026-03-03
**Core Value:** Complete the Kevin Powell JAMstack tutorial CMS setup using Decap CMS, learning each step hands-on.

**Mode:** Guided walkthrough — Claude instructs, user implements.

## v1 Requirements

Requirements for tutorial completion. User performs each step with guidance.

### CMS Setup

- [x] **CMS-01**: User creates /admin folder with index.html loading Decap CMS from CDN — completed Phase 1
- [x] **CMS-02**: User creates config.yml defining Git backend and blog post collection — completed Phase 1
- [x] **CMS-03**: User configures collection fields matching existing frontmatter (title, author, date, tags, image, description, body) — completed Phase 1
- [x] **CMS-04**: User configures media_folder and public_folder paths for image uploads — completed Phase 1
- [x] **CMS-05**: User adds admin folder to Eleventy passthrough copy in eleventy.config.js — completed Phase 1

### Deployment

- [x] **DEP-01**: User deploys site to Netlify from Git repository — completed Phase 2
- [x] **DEP-02**: User enables Netlify Identity on the deployed site — completed Phase 2
- [x] **DEP-03**: User enables Git Gateway in Netlify Identity settings — completed Phase 2
- [x] **DEP-04**: User adds Netlify Identity widget script to site HTML — completed Phase 2
- [x] **DEP-05**: User tests CMS login and creates/edits a post through the admin panel — completed Phase 2
